# XOS iOS App

This directory contains the iOS app for XOS, which runs the Rust engine natively on iOS devices.

## Setup

1. **Build the Rust library for iOS:**
   ```bash
   cd /path/to/xos
   xos build --ios
   # Or manually:
   ./build-ios.sh
   ```

2. **Create Xcode project (if not exists):**
   
   **Option A: Using xcodegen (recommended):**
   ```bash
   brew install xcodegen
   cd ios
   ./create-xcode-project.sh
   ```
   
   **Option B: Manual creation in Xcode:**
   - Open Xcode
   - File > New > Project
   - Choose "iOS" > "App"
   - Product Name: `xos`
   - Interface: Storyboard
   - Language: Swift
   - Save to the `ios/` directory
   - Make sure the project is named `xos.xcodeproj`

3. **Install CocoaPods dependencies:**
   ```bash
   cd ios
   pod install
   ```

4. **Open the workspace in Xcode:**
   ```bash
   open xos.xcworkspace
   ```

5. **Build and run:**
   - Select your target device or simulator
   - Press Cmd+R to build and run

## Project Structure

- `xos/` - Main iOS app source code
  - `AppDelegate.swift` - App entry point
  - `ViewController.swift` - Main view controller with viewport
  - `xos-Bridging-Header.h` - Objective-C bridging header
  - `Info.plist` - App configuration

- `XosModule/` - Swift module for Rust FFI
  - `XosCore.swift` - C FFI bindings for Rust engine
  - `XosViewportView.swift` - Metal-based viewport renderer

- `libs/` - Compiled Rust library (generated by build-ios.sh)
  - `libxos.a` - Universal library for iOS

- `Xos.podspec` - CocoaPods specification
- `Podfile` - CocoaPods dependencies

## Architecture

The iOS app uses:
- **Metal** for GPU-accelerated rendering
- **C FFI** to bridge Swift and Rust
- **CADisplayLink** for 60 FPS rendering loop

The Rust engine runs in a separate module and communicates with Swift through C-compatible functions defined in `src/engine/ios_ffi.rs`.

## Changing Apps

You can change which app runs by:
1. Modifying the `appName` variable in `ViewController.swift`
2. Using the "Change App" button in the UI (if enabled)
3. Available apps: `blank`, `ball`, `tracers`, `camera`, `whiteboard`, `waveform`, `scroll`, `text`, `wireframe`, `triangles`, `cursor`, `audiovis`, `partitions`
